Problem Solving: Subtraction

Objectives:

• Understand the concept of subtraction.
• Learn how to solve subtraction problems step-by-step.
• Practice solving subtraction problems.

Examples:

1. Subtracting Single-Digit Numbers:
8 - 3: 5
7 - 4: 3
2. Subtracting Double-Digit Numbers:
45 - 23: 22
56 - 19: 37
3. Subtracting Triple-Digit Numbers:
234 - 123: 111
578 - 256: 322

Steps to Solve Subtraction Problems:

1. Write down the numbers, aligning the digits by place value.
2. Subtract the digits starting from the rightmost column.
3. If necessary, regroup by borrowing from the next column.
4. Write the difference below the line.
5. Check your work to ensure accuracy.
